JavaScript

Create a file named script.js for the functionality:

javascript
function convertBinary() {
    const binaryInput = document.getElementById('binaryInput').value.trim();
    const resultElement = document.getElementById('result');

    if (binaryInput === '') {
        resultElement.textContent = 'Please enter binary code.';
        return;
    }

    const binaryArray = binaryInput.split(' ');
    let textOutput = '';

    binaryArray.forEach(binary => {
        if (binary.length % 8 !== 0) {
            resultElement.textContent = 'Invalid binary input. Ensure that each binary string is 8 bits long.';
            return;
        }
        
        const decimal = parseInt(binary, 2);
        textOutput += String.fromCharCode(decimal);
    });

    resultElement.textContent = textOutput;
}

How it Works:

  1. HTML: Provides the structure of the page with a textarea for input, a button to trigger conversion, and a p element to display results.
  2. CSS: Adds styling to make the page look nicer.
  3. JavaScript: Contains the logic to convert binary to text. It splits the input by spaces, converts each 8-bit binary string to a decimal, and then to a character.

Usage:

  1. Save these files in the same directory.
  2. Open index.html in your web browser.
  3. Enter binary code into the textarea, where each binary character is separated by a space (e.g., 01001000 01100101 01101100 01101100 01101111 for "Hello").
  4. Click "Convert" to see the text output.

function convertBinary() {
    const binaryInput = document.getElementById('binaryInput').value.trim();
    const resultElement = document.getElementById('result');

    if (binaryInput === '') {
        resultElement.textContent = 'Please enter binary code.';
        return;
    }

    // Split the input by spaces to get individual binary strings
    const binaryArray = binaryInput.split(/\s+/);
    let textOutput = '';

    // Convert each binary string to a character
    for (let binary of binaryArray) {
        if (binary.length !== 8) {
            resultElement.textContent = 'Each binary string should be 8 bits long.';
            return;
        }
        
        // Ensure binary string is valid
        if (!/^[01]+$/.test(binary)) {
            resultElement.textContent = 'Invalid binary input. Only 0s and 1s are allowed.';
            return;
        }

        const decimal = parseInt(binary, 2);
        textOutput += String.fromCharCode(decimal);
    }

    resultElement.textContent = textOutput;
}

Key Improvements:

  1. Binary Input Validation: Ensure each binary string is exactly 8 bits long and contains only 0 and 1.
  2. Split Input by Spaces: Used \s+ to split by any whitespace to handle different spacing styles.
  3. Better Error Messages: Provides clearer feedback on errors.
